# 😋 Grouping Content
```
* <ol>, <ul>, <li>
* <dl>, <dt>, <dd>
* <div>
* <figure>, <figcaption>
* <p>
* <pre>
* <blockquote>
* <main>
* <hr>
```
## <ol>, <ul>, <li>
* `<ol>` : ordered list. 순서가 있는 목록
* `<ul>` : unordered list. 순서가 없는 목록을
* `<li>`는 각 항목들을 나열하는 태그로 list item을 뜻함 
* 주의할 점은 `<li>` 요소는 `<ol>` 혹은 `<ul>` 요소 안에서만 사용되어야 하며,   
  `<ol>`, `<ul>` 의 직계 자식 요소로는 `<li>` 요소만 사용되어야 함
