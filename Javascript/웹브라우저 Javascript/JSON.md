# JSON
* JavaScript Object Notation
* 정보 교환시 사용하기 위해 만들어진 데이터 포맷. JS 언어 문법 빌려 만듦

# JSON과 JS의 차이점
* JSON 사용시 프로퍼티에 반드시 " " 붙여줘야함
* 문자열 표현시 ' ' 사용 불가. 반드시 " "
* JS에서 프로퍼티의 값으로 사용할 수 있는 ```undefined, NaN, Infinity``` 등을 JSON에서는 사용할 수 없음
    * 참고로, JSON은 비록 JS로부터 비롯된 데이터 포맷이지만 그 탄생 목적은 언어나 환경에 종속되지 않고 언제 어디서든 사용할 수 있는 데이터 포맷이 되는 것
    * 따라서 자바스크립트의 문법에서만 유효한 개념을 JSON에서는 나타낼 수 없다는 것은 어찌 보면 당연한 결과

* JSON은 코드가 아니라 데이터 포맷이기 때문에 그 안에 주석을 포함시킬 수 없음
```
JSON.parse
```
* string 타입의 JSON 데이터를 JS 객체로 변환

> 출처 : 코드잇 웹 프론트엔드 스쿨
