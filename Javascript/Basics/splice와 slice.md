# splice 사용시
```
const array = [1, 2, 3, 4, 5];
const result = array.splice(2, 5);
console.log(result); //[3, 4, 5]
console.log(array); //[1, 2]
```

# slice 사용시
```
const array = [1, 2, 3, 4, 5];
const result = array.slice(2, 5);
console.log(result); //[3, 4, 5]
console.log(array); //[1, 2, 3, 4, 5]
```
