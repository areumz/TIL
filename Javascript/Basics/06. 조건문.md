조건문
======
# 조건문(Conditional statement)
* 주어진 조건에 따라서 애플리케이션을 다르게 동작하도록 하는 것

# 조건문의 문법
## If
* if로 시작. if 뒤의 괄호에 조건이 오고, 조건이 될 수 있는 값은 Boolean   
Boolean의 값이 true라면 조건이 담겨진 괄호 다음의 중괄호 구문이 실행

```
if(true){
    alert('result : true');
}
```
* 실행결과 'result : true'   
cf. 괄호에 false가 들어가면 실행되지 x
```
if(true){
    alert(1);
    alert(2);
    alert(3);
    alert(4);
}
alert(5);
```
* 실행결과 1, 2, 3, 4, 5가 차례로 출력

```
if(false){
    alert(1);
    alert(2);
    alert(3);
    alert(4);
}
alert(5);
```
* 실행결과 5만 출력
